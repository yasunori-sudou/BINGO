====================
概要
====================
新年会ビンゴ用のアプリケーションです。
指定の範囲の数字をランダムに選んで溜めていきます。

自作なのでコピーして自由に編集して貰って構いません。

2025.12.22 製作: 須藤

====================
使い方
====================
・アプリの起動
　同ディレクトリにある"bingo.html"をブラウザで開いてください。
　WEBサーバーからアクセス
　http://localhost/nfsys/bingo/bingo.html
　htmlファイルへアクセス
　file:///ダウンロードしたファイルのパス & bingo.html

・抽選
　"抽選" ボタンを押すと、番号が抽選されます。
　抽選された番号はcookieに保存されていますので
　リロードしても値を保持します。

・抽選結果のリセット
　"リセット"ボタンを押すと抽選結果の履歴がリセットされます。

・景品のリスト表示
　JSの配列で景品の設定ができます。
　typeは目玉景品はgold, 普通の景品だとnormalで表示を分けることができます。
　
・景品の強調、消し込み
　景品のテキストをクリックすると当選済みの消込ができます。
　もう一度クリックすると次に当たる景品を強調させることができます。
　強調表示のオールリセット機能はありません。画面リロードしてください。

====================
設定
====================
・ビンゴの最大値はJavaScriptの定数「MAX_NUMBER」を変更してください。
　初期値は75となっています(1～75の数がランダムで選ばれる)
・景品の画像は./img にmain.pngという名前で保存してください。
　抽選の左側に表示されます。

・抽選結果が何列で改行するかの設定は./css/main.cssのgrid-template-columns 
　を変更してください。

・抽選確定時の効果音は"confirm.mp3"となっています。
　効果音を消す場合はcondirm.mp3を無音にするか、JSソースコードの
  playConfirmSound();を消してください。
　※ファイルを消すとJSがエラーになりcookieが保存されません。

====================
使用しているフレームワーク
====================
bootstrap 5.3
jquery- 3.7.1
jsフォルダ / cssフォルダの中にダウンロードされています。

後はHTMLを編集して画像を背景に表示したり自由にやってください。
基本的にボタンと表示領域だけ変更しなければ動きます。
